About Flavour Fusion

A dynamic, interactive recipe experience crafted with passion and code.

The Project

Flavour Fusion is a comprehensive Front-End Engineering Project developed during our B.E. CSE (AI & ML) 2nd Semester. Our objective was to build a highly responsive, user-friendly culinary web application from scratch using modern web technologies.

Instead of a standard static website, we engineered a "Smart Page" architecture. The platform features dynamic URL routing, integrated local video rendering, a real-time mathematical serving calculator, and localized browser storage to seamlessly save user favorites.
